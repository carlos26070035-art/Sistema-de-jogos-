# Sistema-de-jogos-
Olá! me chamo Carlos Eduardo, tenho 17 anos atualmente. hoje eu irei mostrar o Sistema dos jogos usando a linguagem C, Contendo o menu para o acesso aos jogos. inspiração: Dev C++.

Jogo de perguntas: Nesse jogo, o seu objetivo é acertar o máximo de perguntas, sendo no total, 5. as perguntas pode variar de acordo com a dificuldade que o jogador possa ter.

Cobra na Caixa: Nesse jogo, há uma opção dizendo que se o Jogador quisera ver o tutorial, se sim (s) , mostra o tutorial, se não (n ou qualquer tecla). seu objetivo é encontrar um botão para a saida de um templo em que está em uma das 7 caixas alem de que pode ter uma cobra. caso encontre-se com a cobra, você irá perder.

Gousmas war: Nesse jogo, cada jogador começa com 2 gousmas com nivel de fúria 1. caso alguma gousma ultrapasse a fúria 5, ele se desistegra. no jogo, há duas ações, atacar e dividir e caso fique sem uma gousma, se você tiver pelo menos 2 gousmas em sua mão esquerda, você pode trocas para a mão direita.
